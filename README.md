### Description:
Ease Music is an Android music application with sound visualization and beat vibration. Provides convenient search and cache services for music resources.

### Software Usages:
Android:
- Matieral Design
- OkHttp
- Picasso
- Media Player
- SQLite
- YouTube Player

Backend:
- Spring Boot
- YouTube Data API
- yt-dlp
- gRPC
- MySQL
- Docker
